do_map(line='')[source]¶

Apply a function to each item. It can be used as prefix or infix operator.

Usage (prefix)¶

map f x y z

Usage (infix)¶

f x |> map g
f x |> map g $ a b
f x >>= g

If $ is present, then each line from stdin is inserted there. Otherwise each line is appended.
